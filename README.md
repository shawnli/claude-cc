# Claude Code

这个仓库包含 Claude Code 的 Node.js 包及其解压内容。

## 内容

- `anthropic-ai-claude-code-2.0.31.tgz` - Claude Code 的 npm 安装包（版本 2.0.31）
- `package/` - 解压后的包内容

## 关于 Claude Code

Claude Code 是 Anthropic 开发的一个智能编码工具，运行在终端中，能够理解代码库并帮助开发者更快地完成编码任务。

## 安装

```bash
npm install @anthropic-ai/claude-code
```

或者使用本地包：

```bash
npm install ./anthropic-ai-claude-code-2.0.31.tgz
```
